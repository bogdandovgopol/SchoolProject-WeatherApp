# WeatherApp

An Android application that forecasts the weather via DarkSky API. You can save multiple locations using Relm.


# What I Learned

* Basics of Kotlin
* Constraintlayout
* Find user location using MapBox
* Work with API's (google geocache & darksky API) 
* Parse JSON data
* Realm
